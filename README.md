# 🎓 AI-Based Exam Cheating Detection using Webcam & IoT 🚨

> Smart AI-powered online exam monitoring system using **Python, OpenCV, Arduino Uno, and Firebase**

---

## 📌 Project Overview

This project is a **real-time exam cheating detection system** that uses **Artificial Intelligence + Computer Vision + IoT** to monitor students during online examinations.

It detects suspicious activities like:

👀 Looking away from screen  
🚫 Face not visible  
🔄 Unusual head movement  
⚠️ Suspicious behavior

When cheating is detected:

🚨 LED glows  
🔊 Buzzer turns ON  
☁️ Alert sent to Firebase  
📊 Event logged for monitoring
